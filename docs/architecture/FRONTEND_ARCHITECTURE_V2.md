# ENDOS Frontend Architecture V2

ENDOS uses static GitHub Pages as the initial deployment layer.

## Current layer
- HTML pages in `pages/`
- CSS in `assets/css/`
- JavaScript in `assets/js/`
- JSON data in `data/`

## Future layer
The structure is compatible with later migration to:
- React/Vue frontend
- backend API
- graph database
- authentication
- AI-agent orchestration
