# ENDOS – Enterprise Navigation & Decision Operating System

ENDOS ist als Enterprise-Plattform für Navigation, Entscheidungsunterstützung, Evidenzbewertung, Architekturentwicklung und Validierung konzipiert.

## Inhalt

- `index.html` – GitHub-Pages-Startseite
- `assets/` – CSS, JavaScript, Bilder
- `docs/` – Architektur, Roadmap, Changelog
- `repository/` – Wissensobjekte, Patentobjekte, Datenbank, Publikationen
- `releases/` – zukünftige Release-Pakete

## GitHub Pages

Repository öffnen → Settings → Pages → Source: `Deploy from a branch` → Branch: `main` → Folder: `/root` → Save.

Danach ist die Seite erreichbar unter:

`https://<github-user>.github.io/endos/`
